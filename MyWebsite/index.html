    Shakespeare Generator

    This is a simple project that uses TensorFlow and Keras to generate tokens for words using Tokenizer. Following this step, a corpus of words is generated from the dataset, which gives a unique value (index) to each word. Every line in the dataset (poetry) is processed and padded. Labels are separated from the Input sequence texts. The Model has Bidirectional, LSTM layers, a Dropout layer and Dense layers. A visualization of the training accuracy and loss is also shown. , followed by an example with a seed text (random input of sentences) which predicts the most likely next words.
